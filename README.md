# FinBI — Dashboard de Investimentos

Dashboard de investimentos 100% no navegador, com sincronização via GitHub Gist.

## 🚀 Acesso

Após configurar: `https://SEU_USUARIO.github.io/finbi/`

## ⚙️ Setup (15 min)

### 1. Ative o GitHub Pages
- Settings → Pages → Source: **GitHub Actions** → Save

### 2. Crie um Personal Access Token
- https://github.com/settings/tokens/new
- Scope: apenas `gist`
- Copie o token (ghp_...)

### 3. Configure no painel
- Abra o painel → Configurações → GitHub Gist
- Cole o token → Salvar e conectar

## 🔒 Privacidade
- Token salvo APENAS no localStorage do seu browser
- Dados em Gist privado — só você acessa
- Zero servidores de terceiros com seus dados

## 📱 Multi-dispositivo
Use o mesmo token + Gist ID em qualquer dispositivo.
